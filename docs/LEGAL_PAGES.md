# Юридические страницы

## Описание

Созданы страницы Политики конфиденциальности и Условий использования для системы ИФТУТ.

## Созданные страницы

### 1. Политика конфиденциальности (`/privacy`)

**Файл**: `resources/js/Pages/Legal/PrivacyPolicy.vue`

**Разделы:**
1. Какую информацию мы собираем (персональные данные, академическая информация, техническая информация)
2. Как мы используем информацию
3. Защита ваших данных (шифрование, хеширование паролей, контроль доступа)
4. Передача данных третьим лицам
5. Ваши права (доступ, исправление, удаление данных)
6. Использование Cookies
7. Срок хранения данных
8. Защита данных несовершеннолетних
9. Изменения в политике конфиденциальности
10. Контактная информация

**Ключевые моменты:**
- Полное описание сбора и обработки персональных данных
- Меры безопасности (SSL/TLS, bcrypt, контроль доступа)
- Интеграция с SMS-провайдером (OsonSMS)
- Права пользователей на доступ и управление данными
- GDPR-подобная структура, адаптированная для РТ

### 2. Условия использования (`/terms`)

**Файл**: `resources/js/Pages/Legal/TermsOfService.vue`

**Разделы:**
1. Принятие условий
2. Описание сервиса
3. Учетные записи пользователей (безопасность, ответственность)
4. Правила использования системы (разрешенное и запрещенное)
5. Интеллектуальная собственность
6. Контент, размещаемый пользователями
7. Академическая честность (антиплагиат, честное выполнение заданий)
8. Уведомления (email, SMS)
9. Ограничение ответственности
10. Приостановка и прекращение доступа
11. Техническая поддержка
12. Изменения в условиях
13. Применимое право и разрешение споров
14. Контактная информация

**Ключевые моменты:**
- Правила безопасности учетных записей
- Запрет на несанкционированный доступ и вредоносную деятельность
- Академическая честность и запрет плагиата
- Четкие последствия нарушения правил
- Соответствие законодательству Республики Таджикистан

## Интеграция

### Маршруты

Добавлены в `routes/web.php`:

```php
// Юридические страницы (доступны всем)
Route::get('/privacy', function () {
    return Inertia::render('Legal/PrivacyPolicy');
})->name('privacy');

Route::get('/terms', function () {
    return Inertia::render('Legal/TermsOfService');
})->name('terms');
```

**Особенности:**
- Доступны без авторизации
- Не требуют middleware
- Работают как для гостей, так и для авторизованных пользователей

### Страница входа

Обновлен `resources/js/Pages/Auth/Login.vue`:

```vue
<v-btn :href="route('privacy')" ...>
  Политика конфиденциальности
</v-btn>
<v-btn :href="route('terms')" ...>
  Условия использования
</v-btn>
```

**Расположение:** Футер формы входа
